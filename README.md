# lidar-scanner
A script &amp; microcontroller running MicroPython that captures a LIDAR image with a cheap infrared sensor


### Parts
> - 2x micro servo
> - ESP-32 devboard
> - 470 microfarad capacitor
> - blue LED
> - breadboard
> - jumper wires
